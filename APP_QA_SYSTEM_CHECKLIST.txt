Co Pilot Collections Manager — QA Smoke Checklist

After uploading this ZIP:
1. Run npm install && npm run dev.
2. Confirm the polished layout loads, not raw plain HTML.
3. Login as admin.
4. Click System Check and confirm Supabase environment is correct.
5. Confirm account queue opens.
6. Save an agent note.
7. Create a payment plan as employee test user if needed.
8. Confirm admin can edit/delete payment plans.
9. Open Compliance Guard.
10. Open Reports full page and Back to Queue.
11. Open Chat / Team Messenger.
12. Open Pay Desk, Ledger, Tickler, Scorecards.
13. Open Receipts & Letters and generate a receipt.
14. Draw or type an electronic signature and confirm it appears above Authorized Representative.
15. Click Download PDF and confirm the PDF matches the right-side Preview.
16. Confirm the PDF filename includes the document type, consumer, and document number.
17. Confirm no stuck loading or page unresponsive warning.
